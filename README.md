# 形态学学习系统

医学检验形态学学习软件，**二合一**：真题练习 + 细胞标签库。纯静态网页，**无需安装、无需联网、无需服务器**，双击即用。

## 打开方式

- **Windows**：双击 `双击启动.bat`（或直接双击 `index.html`）
- **Mac**：双击 `双击启动.command`（首次若提示未签名：右键 → 打开 → 再点打开）
- 建议用 Chrome / Edge / Firefox 打开。

打开后是主页，点卡片进入任一模块；进入后左侧边栏最上方可一键切换「🏠 主页 / 📝 真题练习 / 🔬 标签库」。

## 两个模块

### 📝 真题练习（`练习系统/`）
- 2024 / 2025 北京市检验医学年会形态学大赛初赛题目，共 **320 道**带图真题。
- 细胞分类图谱浏览、随机抽题、自定义组卷、错题库；可自建题库、编辑内置题。
- **已接入标签库**：分类图谱中有「标签库·细胞图谱」分支，可看图 + 按图快速测验；题库管理可「从标签库选图建题」，一键生成永久选择题。

### 🔬 细胞标签库（`标签库/`）
- 带 9 项标签的形态学图片数据库（编号·来源·分类·子分类·标签·典型程度·疾病·解说），共 **160 张**标注细胞图。
- 可上传图片入库、多维筛选检索、分类管理（增删分类 / 子分类），并从库中自动出题、随机测验。

## 数据与存储

- 所有进度 / 自建内容保存在本地浏览器（localStorage），两个模块各自独立，互不影响。
- 各模块内均可「导出 / 导入」备份，用于换电脑迁移。
- 两模块需保持在同一目录内（即本仓库整体）才能联动。

## 目录结构

```
形态学学习系统/
├── index.html          # 主页（模块入口）
├── 双击启动.command    # Mac 启动器
├── 双击启动.bat        # Windows 启动器
├── 使用说明.txt
├── 练习系统/           # 真题练习模块
└── 标签库/             # 细胞标签库模块
```

## 说明

- 题库与标签库图片为学习 / 比赛素材，部分带有原始来源水印；本仓库仅供个人学习与教学交流使用。
- 2024 题库原文档无标准答案，现用答案为当时参赛者所选、未经官方核对，仅供参考（系统内对存疑题有标记）。
