欢迎使用形态学练习系统

在左侧选择 全部练习、细胞分类 / 组卷 或 错题练习 开始答题；如果新建了自己的题库，可在左上「题库分组」切换。

  • 进度自动保存：练习过程会自动记住做到第几题、答对答错；下次打开在左侧或本页顶部点「▶ 继续上次练习」即可从上次的位置接着做，不用从头开始
  • 内置题库：「2025北京市检验医学年会形态学大赛初赛题目」160 题（答案已校对）、「2024北京市检验医学年会形态学大赛初赛题目」160 题
  • 注意（2024 题库）：原文档无标准答案，现用答案为当时参赛者所选、未经官方核对，仅供参考；多为正确，但 ANA 核型、真菌分种、细胞分化阶段等需以专业判读为准。仅个别有明确缺陷的题保留 ⚠ 标记。
  • 左侧「细胞分类 / 组卷」可按 具体细胞名称逐层浏览（如 血细胞·骨髓 → 红系 → 早幼红细胞）、看图、按类练习、随机抽题，或勾选小类（支持全选）自定义组卷
  • 已接入「细胞标签库」：分类图谱中新增「🧫 标签库·细胞图谱」分支——点「看图」查看某种细胞的全部标注图片（含典型程度、形态解说），点「练习」/勾选「组卷」即按图快速测验
  • 「题库管理 → 🔬 从标签库选图建题」：勾选标签库图片即可一键生成永久选择题并存入指定题库（自动配同系列干扰项），生成的题目同时并入分类图谱
  • 已校对答案：少量存在拼写错误、术语争议或选项设计歧义的题目，会显示 ⚠ 答案存疑 红色徽章，点击徽章查看说明
  • 选项支持点击作答，单选自动判分，多选需点击「提交答案」；答完会显示正确 / 错误及正确答案
  • 答错的题目会自动进入「错题库」（跨分组通用）
  • 答题时可使用右侧「题号导航」或「跳转到第 __ 题」快速切换
  • 左侧「界面尺寸」可一键适配 13.6″ / 15.6″ / 27″ 屏幕，或点「自动适配当前屏幕」精确缩放
  • 可通过左侧「题库管理」新建题库，或直接编辑内置题库（改题干/选项/答案、加题、删题）；编辑 2024 题时清空「存疑标记」即可去掉待核对红标
  • 自建题库可「导出」为文件，在另一台电脑上「导入」即可迁移与朋友共享
